# Exploratory Data Analysis of Heart Disease Dataset

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a heart disease dataset to understand patterns, relationships, and important health factors affecting heart disease.

## 📊 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Steps Performed
- Data Loading
- Data Cleaning
- Handling Missing Values
- Data Visualization
- Finding Insights

## 📈 Key Insights
- Age plays an important role in heart disease
- High blood pressure increases risk
- Certain features show strong correlation with disease
